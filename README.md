# BatchImage

A privacy-first, browser-based bulk image processing tool. Compress, resize, convert, and watermark hundreds of images without uploading them to any server.

🌐 **Live demo**: [https://batchimage.vercel.app](https://batchimage.vercel.app) *(replace with your own URL after deploying)*

![BatchImage Screenshot](./screenshot.png)

## ✨ Features

- **Batch compression** – shrink JPEG/PNG/WebP files in one click
- **Resize** – set max width and/or height
- **Format conversion** – convert to JPEG, PNG, or WebP
- **Watermark** – add text watermark with adjustable opacity and position
- **Privacy first** – all processing happens locally in your browser
- **Free plan** – process up to 5 images per batch
- **Pro plan** – process up to 500 images per batch for $5/month

## 🛠️ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/image-batch-pro.git
cd image-batch-pro
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## 📦 Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the project
3. Click **Deploy**
4. Done
